Verfügbare Commands

  • /help - Zeigt eine Liste aller verfügbaren Befehle an.
  • /kick [Benutzer] - Kickt einen Benutzer vom Server.
  • /ban [Benutzer] - Bannt einen Benutzer vom Server.
  • /mute [Benutzer] - Stummschaltet einen Benutzer.
  • /unmute [Benutzer] - Hebt das Stummschalten eines Benutzers auf.
