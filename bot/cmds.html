Verfügbare Commands

  • /help - Zeigt alle verfügbaren Befehle
  • /info - Zeigt Infos über den Bot
  • /kick [user] - Kickt einen Benutzer vom Server
  • /ban [user] - Bannt einen Benutzer vom Server
  • /mute [user] - Stummschaltet einen Benutzer
  • /clear [amount] - Löscht eine angegebene Anzahl von Nachrichten
  • /announce [message] - Sendet eine Ankündigung
  • /ping - Zeigt die Reaktionszeit des Bots an
