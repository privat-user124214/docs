Verfügbare Bot Commands

Hier findest du eine Übersicht aller verfügbaren Commands für den Novarix Bot.

  • /help - Zeigt alle verfügbaren Befehle an.
  • /info - Zeigt Informationen über den Bot.
  • /kick [user] - Kickt einen Benutzer vom Server.
  • /ban [user] - Bannt einen Benutzer vom Server.
  • /mute [user] - Stummschaltet einen Benutzer.
  • /clear [amount] - Löscht eine angegebene Anzahl von Nachrichten.
  • /announce [message] - Sendet eine Ankündigung im Server.
  • /ping - Zeigt die Reaktionszeit des Bots an.
